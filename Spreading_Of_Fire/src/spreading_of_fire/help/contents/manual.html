User Manual

Installation

For user

  • The Java must be installed, either JDK or JRE. We recommend Java 1.8 as it is the version that we used.
  • Open "Spreading_Of_Fire.jar"

For Developer

  • We recommend Netbeans IDE 8.0.1 with JDK 8u25.
  • Open Netbeans IDE then File -> Open Project ...
  • Select folder "Spreading_Of_Fire".
  • The source code was separated in 7 parts
    • Spreading_Of_Fire : The class that will initially run, used for running the main class
    • Main : The class that will load the Model, View, and Controller.
    • Model : The core of the application, every functions of the application are in here.
    • Controller : The input of the application, it will control and config the parameter of Model.
    • View : The output of the application, it will receive data from Model and represent it.
    • Cell : The class that contains the cell properties.
    • Help : The help window that appears when you push the help button in controller

User Interface



Left Panel

This panel shows the simulation of spreading of fire

Symbol Value Meaning
Yellow Plain 0 Empty
Green Oval 1 Tree
Red Oval 2 Fire
Blue Oval 2 Fire that burned by Lightning

Step
The number of times that the fire spread. The step count starts with 0.

Show Value
The status of showing values of the simulation.

  • true - Show both values and symbols.
  • false - Hide the values and show only symbols.

Lightning Status
The status of showing how the lightning cells burn.

  • true - The cell will take 5 step to become an empty cell at step 6. The cell will spread fire in step 2, 3, and 4, and not spread fire int step 1, and 5.
  • false - The cell will immediately become an empty cell after it was burned. In other word, the cell will act like usual fire cell.

Remaining Tree
The percentage of the trees left from the beginning. The value is calculated by (The current number of trees)/(the initial number of trees) x100%.



Right Panel

This panel show the controller of factors of the simulation

Button

  • Spread - Spread the fire step by step. By pressing this button, the step count will increase by 1.
  • Auto Spread - Spread the fire respectively. By pressing this button, the step count will continually increase by 1.
  • Stop - Stop the spreading process. By pressing this button, if Auto Spread was pressed, the step counting will be stopped.
  • Reset - Reset the field. By pressing this button, the field will be reset and the step count will become 0.
  • Toggle Value - Toggle the status of showing values of the simulation. By pressing this button, the field will show or hide the value alternatively.
  • Step Lightning - Toggle the status of step lightning of the simulation. By pressing this button, the Lightning status in the view will toggle the value alternatively. The meaning of value can be read above.

Slider

  • Fire Catch Rate [0-100] : The chance that fire can spread by catching the neighbor tree. The default value is 50%.
  • Tree Density [0-100] : The density of tree at the beginning or at the step 0. The default value is 100%.
  • Initial Burning [0-100] : The chance that the initial tree is burning. The default value is 0%.
  • Size [11x11 - 99x99] : The size of the field, which means the number of element for each rows and columns. The default value is 31x31.
  • Delay [1-1000] : The time delay for each step count in milliseconds. The default value is 100 ms.
  • Lightning Chance [0-100] : The chance that each trees in the field will get lightning. The default value is 0%.
  • Wind Direction [North, West, East, South] : The direction of the wind. The default value is North.
  • Wind Level [0-2] : The level of the wind. The default value is 0.


Level 0 - None : There is no wind. The fire will spread in four direction normally.


Level 1 - Low : The fire will spread in four direction normally. If it spreads successfully, it will have a chance to spread 1 additional cell in the same direction as wind direction, in the sample image is North


Level 2 - High : The fire will spread in four direction normally. If it spreads successfully, it will have a chance to spread 2 additional cells in the same direction as wind direction, in the sample image is North

  • Initial Fire X & Y Coordinate [1 - size of field] : The coordinate of the first fire cell. The default value is center of field, or (16,16) (The field is 31x31 by defalut). When the size of the field change, the value will be changed into the center of the field.
