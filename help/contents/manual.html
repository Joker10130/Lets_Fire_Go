User Manual

User Interface



Left Panel

This panel shows the simulation of spreading of fire

Symbol Value Meaning
Yellow Plain 0 Empty
Green Oval 1 Tree
Red Oval 2 Fire

Step
The number of times that the fire spread. The step count starts with 0.

Show Value
The status of showing values of the simulation.

  • true - Show both values and symbols.
  • false - Hide the values and show only symbols.


Right Panel

This panel show the controller of factors of the simulation

Button

  • Spread - Spread the fire step by step. By pressing this button, the step count will increase by 1.
  • Auto Spread - Spread the fire respectively. By pressing this button, the step count will continually increase by 1.
  • Stop - Stop the spreading process. By pressing this button, if Auto Spread was pressed, the step counting will be stopped.
  • Reset - Reset the field. By pressing this button, the field will be reset and the step count will become 0.
  • Toggle Value - Toggle the status of showing values of the simulation. By pressing this button, the field will show or hide the value alternatively

Slider

  • Fire Catch Rate [0-100] : The chance that fire can spread by catching the neighbor tree. The default value is 50%.
  • Tree Density [0-100] : The density of tree at the beginning or at the step 0. The default value is 100%.
  • Initial Burning [0-100] : The chance that the initial tree is burning. The default value is 0%.
  • Size [11x11 - 99x99] : The size of the field, which means the number of element for each rows and columns. The default value is 31x31.
  • Delay [1-1000] : The time delay for each step count in milliseconds. The default value is 100 ms.
